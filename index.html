Discover · Definition

You just found a sequence!

Sequence (ลำดับ)
A sequence is a function whose domain is the set {1, 2, 3, …, n} or the set of all positive integers {1, 2, 3, …}, arranged in increasing order starting from 1.
Finite sequence

Domain = {1, 2, 3, …, n}
Has a definite last term

Infinite sequence

Domain = {1, 2, 3, …}
Goes on forever — no last term

Critical rule
The domain must start at 1 and be consecutive. A function with domain {2, 4, 6, 8} is NOT a sequence because it skips 1, 3, 5. If no domain restriction is stated, we assume the sequence is infinite.

Quick rule
If the domain ends at a specific value (e.g. n ∈ {1…5} or the list shows a last term), it's finite. If it uses "…" with no stated end, it's infinite.

Summary so far
✓ Domain starts at 1 AND is consecutive → Sequence
✓ Ends at n → Finite · continues forever → Infinite
✗ Domain skips numbers, doesn't start at 1, or uses negatives → Not a sequence
